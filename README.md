## CLI Utility

### Install

Install [Deno](https://deno.land/) and run the following command:

```shellscript
deno install https://mandel59.github.io/bonji-input/siddhamconv.js
```

### Usage

```shellscript
echo "siddha;m" | siddhamconv > out.txt
```
